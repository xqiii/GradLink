# GradLink - 中国地区毕业同学去向应用

## 项目简介
GradLink 是一个专门用于展示中国地区毕业同学去向的应用平台。该项目旨在帮助校友、在校生和教育机构更好地了解毕业生的就业趋势、行业分布和地理流向，为职业规划和教育资源优化提供有价值的参考。

注意：该项目完全使用trae开发，不涉及任何商业用途。

## 项目结构
- **backend/**: 后端服务代码
- **frontend/**: 前端应用代码
- **config/**: 配置文件目录
- **docs/**: 项目文档
